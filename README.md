**Senior Full Stack Developer** with **8+ years** of experience designing, building, and shipping production web applications end to end — from responsive UIs and APIs to databases, integrations, and deployments. I'm also **proficient in CMS & commerce development** — **Shopify**, **WordPress**, **Webflow**, **Joomla**, **Payload CMS**, **Storefront API**, and other platforms (Drupal, Magento, headless CMS, and more).

This is my **new GitHub profile**, where I'm consolidating selected work, experiments, and portfolio projects. My contribution graph here is still growing; my professional experience spans years of delivery across product teams, code reviews, mentoring, and production ownership.

- End-to-end ownership — requirements → architecture → implementation → release
- Scalable **REST APIs**, clean service layers, and maintainable frontends
- Strong focus on **TypeScript**, **React**, and **Node.js** ecosystems
- **CMS & e-commerce** — Shopify, WordPress, Webflow, Joomla, Payload, Storefront API, and more

- Collaboration with product, QA, and DevOps in agile environments
